# 电化学测试数据分析与等效电路拟合工具 - PRD

## 1. 产品概述
电化学测试数据分析工具，面向电化学研究人员和工程师，提供循环伏安(CV)、电化学阻抗谱(EIS)、恒流充放电等电化学数据的可视化分析与等效电路拟合功能。帮助研究人员快速量化关键电化学参数，评估材料性能与循环稳定性。

**核心价值**：
- 一站式多维度电化学数据分析，无需专业软件即可完成数据处理
- 智能识别氧化还原峰、自动拟合等效电路，提升分析效率
- 多条件数据横向对比，辅助量化温度、浓度等因素影响

## 2. 核心功能

### 2.1 用户角色
| 角色 | 使用方式 | 核心权限 |
|------|---------|---------|
| 研究人员 | 浏览器直接访问 | 上传数据、分析图表、导出结果 |
| 学生用户 | 浏览器直接访问 | 所有分析功能、查看示例数据 |

### 2.2 功能模块
1. **数据导入模块**：支持CSV/TXT格式数据导入，支持多文件批量上传
2. **循环伏安(CV)分析**：峰识别、峰参数标注、多循环叠加对比
3. **电化学阻抗谱(EIS)分析**：Nyquist图、Bode图、等效电路拟合
4. **恒流充放电分析**：容量计算、能量密度、循环衰减曲线
5. **多条件对比**：不同温度/浓度数据横向对比分析

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|------------|
| 首页 | 导航与概览 | 功能入口卡片、快速上传、示例数据入口 |
| CV分析页 | CV曲线展示 | 交互式CV曲线图、峰识别结果面板、峰参数表 |
| EIS分析页 | EIS图表与拟合 | Nyquist图、Bode图、等效电路选择、拟合参数表、残差分析 |
| 充放电分析页 | 充放电曲线 | 放电曲线、容量计算、能量密度、循环衰减曲线 |
| 对比分析页 | 多条件对比 | 数据分组、对比图表、参数对比表 |

## 3. 核心流程

```mermaid
flowchart LR
    A["用户上传数据文件"] --> B["选择分析类型"]
    B --> C{"数据解析与校验"]
    C -->|CV数据| D["CV分析：峰识别 + 多循环对比"]
    C -->|EIS数据| E["EIS分析：Nyquist/Bode图 + 等效电路拟合"]
    C -->|充放电数据| F["充放电分析：容量/能量密度 + 循环衰减"]
    D --> G["结果可视化与参数导出"]
    E --> G
    F --> G
    G --> H["多条件数据对比分析"]
```

**主流程说明**：
1. 用户上传CSV/TXT格式的电化学测试数据文件
2. 系统自动识别数据类型或用户手动选择分析模式
3. 数据解析后自动进行基线校正和预处理
4. 根据分析模式执行对应分析算法（峰识别/拟合/容量计算）
5. 结果以交互式图表展示，参数以表格形式呈现
6. 支持多组数据横向对比和结果导出

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝科技风，深蓝色(#1e3a5f) 搭配 青色点缀(#00d4aa)
- **辅助色**：浅灰背景、深灰文字、数据系列色(橙、绿、紫、红)
- **设计语言**：科研仪器风格，精密感与科技感并重
- **按钮风格**：圆角矩形按钮，微立体阴影，悬停微动效
- **字体**：标题使用现代无衬线字体，正文清晰易读
- **布局风格**：左侧导航 + 主内容区卡片式布局
- **图标风格**：线性简洁科学仪器风格图标

### 4.2 页面设计总览
| 页面名称 | 模块名称 | UI元素 |
|---------|---------|--------|
| 首页 | 功能概览 | 顶部导航、大标题、功能卡片网格、上传区域、示例数据入口 |
| CV分析页 | CV曲线分析 | 左侧文件列表、主图表区、峰参数面板、多循环切换 |
| EIS分析页 | EIS拟合 | 双图布局(Nyquist+Bode)、等效电路选择器、参数表格、残差图 |
| 充放电分析页 | 充放电分析 | 放电曲线图、容量计算卡片、循环衰减图、能量密度展示 |
| 对比分析页 | 多条件对比 | 数据分组管理、对比图表、参数对比矩阵 |

### 4.3 响应式
- 桌面端优先设计，兼顾平板适配
- 图表区域自适应宽度
- 移动端简化导航折叠菜单
